# Survey App

A simple survey web app built with Flask for collecting lifestyle data.

## Features

- Collect user info and lifestyle preferences
- Save data to SQLite
- View calculated results like:
  - Total submissions
  - Age statistics
  - Food preferences
  - Rating averages

## Tech Stack

- Python 3
- Flask
- SQLite
- HTML + CSS

## How to Run

```bash
pip install -r requirements.txt
python app.py
